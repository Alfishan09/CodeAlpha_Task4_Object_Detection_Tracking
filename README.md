# 🎯 Real-Time Object Detection and Tracking using YOLOv8

## 📌 Project Overview
This project performs real-time object detection and multi-object tracking using YOLOv8, OpenCV, and ByteTrack. The system captures live video from a webcam, detects multiple objects, and tracks them across frames using unique tracking IDs.

## ✨ Features
- Real-time object detection
- Multi-object tracking with unique IDs
- Live webcam integration
- Bounding box visualization
- Object labels and confidence scores
- YOLOv8 pre-trained model
- ByteTrack tracking algorithm

## 🛠️ Technologies Used
- Python
- OpenCV
- Ultralytics YOLOv8
- ByteTrack
- Streamlit

## ⚙️ Installation
```bash
pip install -r requirements.txt
```

## ▶️ Run the Project
```bash
streamlit run app.py
```

## 📄 Output
The system opens the webcam, detects objects in real time, draws bounding boxes, assigns tracking IDs, and tracks object movement across video frames.
